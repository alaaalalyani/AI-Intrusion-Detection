# AI-Based Intrusion Detection System

## Overview
This project uses machine learning to detect cyber attacks in network traffic.

## Approach
- Data preprocessing
- Binary classification (Normal vs Attack)
- Random Forest model

## Results
The model achieved high accuracy in detecting malicious traffic.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
## How to Run
1. Install required libraries:
   
``` bash
pip install pandas numpy scikit-learn matplotlib
```

2. Open the notebook file:
intrusion_detection.ipynb

3. Run all cells to see the results.
